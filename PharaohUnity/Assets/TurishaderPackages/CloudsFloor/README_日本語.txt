# スタイライズドクラウドシェーダー（非ボリューム）

このシェーダーは、頂点変位を使用してスタイライズされた非ボリュームの雲をシミュレートするように設計されています。海洋、空、またはシュールな風景などの環境に最適化されています。以下に、各公開パラメータの詳細な説明を示し、求める視覚的な結果を得るのに役立ててください。

## マテリアルパラメータ

🌥️ CloudHeight (float)
雲層の基準高さを定義します。ワールドまたはローカル空間で雲を上下させるのに便利です。

🎚️ Contact fade (float)
雲が地面や他のジオメトリに近づいたときにどの程度フェードするかを制御します。値が高いほど、より滑らかな移行が作成されます。

🎨 DarkColor (Color)
これは雲の影や暗い部分の色です。深みと視覚的なコントラストを追加します。

☀️ LightColor (Color)
光の下での雲の主な色です。通常、明るく、日光が当たる部分です。

🌓 DarkPower (float)
暗さや影の効果の強さを制御します。値が高いほど、コントラストとボリューム感が増します。

🌬️ TurbulenceSpeed (float)
時間とともに雲の形状がどれくらい速く移動または変化するかを制御します。生命感と雰囲気を与えます。

🔍 Scale (float)
雲の形状を生成するために使用されるノイズやテクスチャパターンのスケールを決定します。値が低いと、より大きく、柔らかい雲が作成されます。値が高いと、より多くの詳細が追加されます。
