Thank you for purchasing our environment pack!
This package has been designed for the Universal Render Pipeline (URP). If you notice any shader errors (pink surfaces), 
please make sure that your project is setup for URP and that the Shader Graph package has been installed from the Package Manager.

Additional Setup:
To provide a better visual appareance of the water, Opaque Texture needs to be enabled in your Universal Render Pipeline Asset.To do so go to Edit->Project Settings->Quality and double click the Render Pipeline Asset, which should then be selected in the Inspector. Make sure that Opaque Texture is enabled.

Performance Tips:
Depending on your systems hardware and your pipeline settings you might notice performance issues when playing our demo scene, as it uses quite a dense amount of foliage.
If you are facing performance problems we recommend to disable MSAA in the Quality Settings of your Universal Render Pipeline Asset.
You might also want to reduce the Detail Distance and Detail Density Scale in the settings of the Terrain in the demo scene as this will reduce the amount of foliage and provide a better performance depending on your systems GPU hardware.
